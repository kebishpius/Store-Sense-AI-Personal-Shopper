# Store-Sense-AI-Personal-Shopper

## Core Concept

A real-time, interruptible AI shopping assistant that:

* 👁️ Sees products through camera (barcode, label, shelf tag)
* 🗣️ Talks naturally with the user
* 💰 Tracks prices across stores
* 🥗 Analyzes nutrition
* ⭐ Pulls reviews
* 📉 Advises whether to buy now or wait

**Powered by:**

* Gemini Live API (real-time multimodal conversation)
* ADK tools (Search, custom pricing tools)
* Google Cloud backend
